== ルール説明

1. 2人一組のチームを作る
2. くじの結果をから、パートナーの服装をチェックする
3. パートナーの服装が、くじに一致したらチェックした人がポイントゲット

用意したくじには、チェックする対象を指示するのも、色や柄を指示するものの2種類
 チェック対象には、上着・パンツ・ひざ下・アクセサリーの4種類
 色や柄には、赤・茶・青・黄・緑・黒・グレー・白・アロハ・沖縄っぽいものの計10種類
 中にはワイルドカードのような複数の色を指定するようなものも含まれている
通常はパーツと色の組み合わせが一致すると1ptゲット!!
 例1) 【上着 • 赤】の場合、パートナーが赤っぽい色のTシャツや上着を着ていれば1ptゲット!!
柄物(アロハ・沖縄っぽいもの)は2ptゲット!!
 例2) 【上着 • アロハ】の場合、パートナーがアロハっぽい柄のTシャツや上着を着ていれば2ptゲット!!
