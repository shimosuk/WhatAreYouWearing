== ルール説明

2人一組のチームを作る
お互いにファッションチェックしてもらう（ドン小西のような厳しいチェックは不要）
用意したくじには、チェックする対象を指示するのも、色や柄を指示するものの2種類

くじを引き、その結果を得て、パートナーの服装をチェックする
パートナーの服装が一致したらチェックした人がポイントゲット

チェック対象には、上着・パンツ・ひざ下・アクセサリーの3種類
色や柄には、赤・茶・青・黄・緑・黒・グレー・白・アロハ・沖縄っぽいものの系10種類
通常はパーツを色の組み合わせが一致すると1ptゲットするが、柄物(アロハ・沖縄っぽいもの)は2ptとする
中にはワイルドカードのような複数の色を指定するようなものも含まれている
