== ルール説明

1. 2人一組のチームを作る
2. 2種類(チェックポイントと色)のくじを引き、その組み合わせで、パートナーの服装をチェックする
3. パートナーの服装が、くじに一致したらチェックした人がポイントゲット

 チェック対象には、上着・パンツ・ひざ下・アクセサリーの4種類
 色や柄には、赤・茶・青・黄・緑・黒・グレー・白・アロハ・沖縄っぽいものの計10種類
 柄(アロハ・沖縄っぽい)で一致した場合は、2ptゲット(通常は1pt)
 中にはワイルドカードのような複数の色を指定するようなものも含まれている

== 進行例

1. くじを引く
2. 【上着 • 白】の組み合わせがでた
3. お互いの服をチェックしあう
4. パートナーが白っぽい上着を着ていれば、1ptゲット

 <補足>
 柄(アロハ・沖縄っぽい)で一致した場合は、2ptゲット
